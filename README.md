🖐️ Real-Time Hand Gesture Recognition
Project Overview
This is a Computer Vision project that recognizes hand gestures in real-time via a webcam. It uses a Convolutional Neural Network (CNN) to identify gestures like "Palm," "Thumb," and "OK," and features a web interface built with Streamlit.

How to Run
Install dependencies: pip install streamlit tensorflow opencv-python

Run the app: streamlit run gesture_app.py

Place your hand inside the blue box on the screen.

Technologies Used
TensorFlow/Keras (Deep Learning)

Streamlit (Web Framework)

OpenCV (Camera handling and Image Processing)
